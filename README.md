# FirstGame

Jogo 2D feito em Python com Pygame.

## Controles
- Space: Saltar
- CTRL: Atirar

## Como jogar
1. Clone o repositório ou baixe o ZIP.
2. Instale Python 3.x e a biblioteca Pygame.
3. Execute `python main.py`.
